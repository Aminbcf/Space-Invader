\hypertarget{md_README_autotoc_md33}{}\doxysection{\texorpdfstring{Vue d\textquotesingle{}Ensemble}{Vue d\textquotesingle{}Ensemble}}\label{md_README_autotoc_md33}
{\bfseries{Space Invaders MVC}} est une implémentation moderne du jeu d\textquotesingle{}arcade classique Space Invaders, développée en langage C (norme C11) selon l\textquotesingle{}architecture Modèle-\/\+Vue-\/\+Contrôleur (MVC). Le projet offre deux interfaces utilisateur distinctes \+:


\begin{DoxyItemize}
\item {\bfseries{Interface graphique SDL3}} \+: Version complète avec graphismes, textures, polices TTF et support d\textquotesingle{}images
\item {\bfseries{Interface texte ncurses}} \+: Version terminal légère pour une compatibilité maximale
\end{DoxyItemize}\hypertarget{md_README_autotoc_md34}{}\doxysubsection{\texorpdfstring{Caractéristiques Principales}{Caractéristiques Principales}}\label{md_README_autotoc_md34}

\begin{DoxyItemize}
\item Architecture MVC stricte avec séparation claire des responsabilités
\item Double implémentation de vue (SDL3 et ncurses) partageant le même modèle
\item Code modulaire et réutilisable
\item Gestion robuste de la mémoire (vérifiée avec Valgrind)
\item Compilation séparée pour chaque interface
\item Support complet sur Linux (Debian/\+Ubuntu et distributions dérivées)
\end{DoxyItemize}
